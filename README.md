## Quick Start

### Installation

```bash
# Install dependencies
npm install

# Start the server
npm start
```

The application will be available at `http://localhost:3000`

### Development Mode

```bash
# Start with auto-reload
npm run dev
```

## Usage

1. Fill in your personal information
2. Add your work experience, education, skills, and languages
3. Click "Preview CV" to see your resume
4. Choose a theme from the dropdown
5. Click "Download PDF" to save your CV
